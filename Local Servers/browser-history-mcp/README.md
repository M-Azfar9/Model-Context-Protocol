# browser-history-mcp

A FastMCP server that provides searchable, read-only access to your local browser history and bookmarks.

## Features
- Search Chrome (and optionally Firefox) browser history by keyword or URL
- Query recent browsing activity by time range
- Identify frequently visited domains
- Search bookmarks
- Summarize previously visited pages (title + URL metadata)

## Setup
Managed via `uv` — no manual venv needed.

## Usage
Integrated with Claude Desktop via stdio transport.
